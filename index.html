The Parallel and Distributed Computing Research Lab (PDCRL) in the Department of CSE at IIT Hyderabad engages in a wide range of theoretical and experimental research in the context of Software Transactional Memory, Parallel and Concurrent Computing, Concurrent Data structures.

Our research addresses the challenges in parallel and concurrent computing using a three-way approach consisting of shared-memory programming techniques, algorithm design, and performance analysis for a broad class of irregular problems in concurrent computing. We are designing efficient concurrent protocols using optimised concurrent data structures like chained hash table using lazy lists, graphs etc. Recently we have also started exploring concurrency enhancement using the semantics of the transactional methods and the underlying shared data structures.


The broader areas of interest are as follows:

  • Parallel Computing
  • Concurrent Data Structures
  • Software Transactional Memory
  • Distributed Systems
